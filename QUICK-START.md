## 🎨 Customization Quick Tips

### Change Phone Number:
Search & replace in `index.html`:
- `9356514611` → Your number
- `9403668234` → Your alternate number

### Change WhatsApp:
Replace all instances of:
- `wa.me/919356514611` → Your WhatsApp

### Change Colors:
Edit CSS `:root` section in `index.html`:
```css
--primary: #2563eb;    /* Main color */
--accent: #10b981;     /* Accent color */
```
